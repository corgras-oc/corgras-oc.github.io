Документация Текстовый блок HTML PLUS

Позволяет эффективно управлять скриптом «Подробнее», определять, на каких экранах он должен функционировать, настраивать уровни заголовков, создавать собственные шаблоны, управлять доступом к контенту, легко вставлять изображения и видео, а также регулировать расположение контента относительно изображений или видео. Это делает модуль мощным инструментом для создания гибкого и интуитивно понятного контента на вашем веб-сайте.

Установка модуля

Чтобы установить модуль, выполните следующие шаги:

  • Войдите в Админ-панель и перейдите в раздел Меню -> Расширения -> Установка расширений;
  • Загрузите и установите архивный файл HTML_Content_Plus_x_OpenCart_x.ocmod.zip;
  • После успешной установки перейдите в раздел Меню -> Система -> Группы пользователей и установите права доступа для администратора (в разделах «Разрешить просмотр» и «Разрешить редактирование» выберите extension/module/cs_html_content_plus).
  • Завершив эти шаги, перейдите в раздел Меню -> Расширения -> Модули и установите модуль Текстовый блок HTML PLUS.

Настройка модуля

Для настройки модуля перейдите в раздел Меню -> Расширения -> Модули -> Текстовый блок HTML PLUS

Основные параметры настройки:

  • Название модуля - отображаемое название в админ-панели;
  • Статус - активация / деактивация модуля;
  • Изображение - вставка изображений для разных экранов или одного для всех;
  • Видео список - выбор источника видео;
  • Ссылка на видео - для поддерживаемых сервисов, просто вставьте ссылку на видео, для других - скопируйте ссылку из iframe src="".

Дополнительные параметры настройки:

  • Позиция контента - расположение текста относительно изображения или видео;
  • Уровень заголовка - уровень заголовка для отображения на сайте;
  • Статус скрипта «Подробнее - включение / выключение скрипта;
  • Высота видимого текста - какая часть текста будет видимой, остальная часть будет скрыта;
  • Разрешение экрана - на каких разрешениях экрана будет работать скрипт;
  • Изменить размер большого изображения - включение изменения размера собственных / выключение для отображения исходного размера;
  • Размер большого изображения (Ш x В) - для больших экранов;
  • Изменить размер среднего изображения - включение изменения размера собственных + если изображение для средних экранов не добавлено, оно будет создано из большого. Выключение для отображения исходного размера;
  • Размер среднего изображения (Ш x В) - для средних экранов;
  • Изменить размер маленького изображения - включение изменения размера собственных + если изображение для маленьких экранов не добавлено, оно будет создано из большого. Выключение для отображения исходного размера;
  • Размер маленького изображения (Ш x В) - для маленьких экранов.

Доступ:

  • Доступ - кто увидит этот блок: для всех пользователей / только для зарегистрированных / только для гостей;
  • Группы покупателей - управление доступом к группам клиентов.

Дизайн:

  • Шаблон - после создания шаблона для модуля, вы можете выбрать его здесь.

Стилизация

Шаблоны:

Для настройки вида блоков с описанием создайте собственный шаблон для вашего модуля и разместите его в папке:

catalog/view/theme/default/template/custom_templates/module/html_content_plus/имя_файла_латинскими_буквами.twig.tpl

В качестве альтернативы, вы можете использовать стандартный шаблон по следующему пути:

catalog/view/theme/default/template/extension/module/cs_html_content_plus.twig.tpl

CSS-стилизация:

Если стандартный дизайн кнопки «Развернуть / Свернуть» не соответствует вашему дизайну сайта, вы можете внести изменения в CSS стили в файле cs_html_content_plus.min.css, который находится по следующему пути:

catalog/view/javascript/cs_libs/css/cs_html_content_plus.min.css
