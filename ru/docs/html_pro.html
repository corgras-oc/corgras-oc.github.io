Документация Текстовый блок HTML PRO

Позволяет создавать и настраивать сетки текстовых блоков на вашем веб-сайте. С помощью этого инструмента вы можете легко создавать неограниченное количество текстовых блоков и настраивать их в соответствии с вашими потребностями и дизайном. Полная гибкость и контроль над размещением и содержанием текста на вашем веб-сайте.

Установка модуля

Чтобы установить модуль, выполните следующие шаги:

  • Войдите в Админ-панель и перейдите в раздел «Меню -> Расширения -> Установка расширений»;
  • Загрузите и установите архивный файл «HTML_Content_Pro_x_OpenCart_x.ocmod.zip»;
  • После успешной установки перейдите в раздел «Меню -> Система -> Группы пользователей» и установите права доступа для администратора (в разделах «Разрешить просмотр» и «Разрешить редактирование» выберите «extension/module/cs_html_content_pro»).
  • Завершив эти шаги, перейдите в раздел «Меню -> Расширения -> Модули» и установите модуль «Текстовый блок HTML PRO».

Настройка модуля

Для настройки модуля перейдите в раздел «Меню -> Расширения -> Модули -> Текстовый блок HTML PRO»

Основные параметры настройки:

  • Название модуля - название, которое будет отображаться в админ-панели;
  • Статус - статус модуля отвечает за отображение модуля на сайте.

Раздел:

  • Изображение - вставка изображений для различных размеров экранов или одно для всех;
  • Видео список - выбор, откуда будет вставлено видео;
  • Ссылка на видео - для поддерживаемых сервисов достаточно вставить любую ссылку на видео, чтобы другие могли скопировать ссылку с « iframe src="" ».

Блоки:

  • Изображение - вставка изображений для различных размеров экранов или одно для всех;
  • Class иконки - для отображения иконки добавьте ее CLASS. Смотрите документацию к подключенной библиотеке;
  • Видео список - выбор, откуда будет вставлено видео;
  • Ссылка на видео - для поддерживаемых сервисов достаточно вставить любую ссылку на видео, чтобы другие могли скопировать ссылку с « iframe src="" »;
  • Позиционирование медиа-элементов - размещение медиа-элементов относительно текста;
  • Категория - ссылка на категорию;
  • Статья - ссылка на информационную статью;
  • Производитель - ссылка на производителя;
  • Другая ссылка - любая другая ссылка;
  • Название кнопки - текст, который будет отображаться на кнопке;
  • Class - индивидуальный CSS класс для блока;
  • Id - индивидуальный CSS идентификатор для блока;
  • Атрибуты - индивидуальный HTML атрибут для блока.

Настройки:

  • Сетка контента - в каком порядке будут отображаться блоки;
  • Class - индивидуальный CSS класс для раздела;
  • Id - индивидуальный CSS идентификатор для раздела;
  • Атрибуты - индивидуальный HTML атрибут для раздела;
  • Колонки - настройка количества колонок для разных экранов;
  • Выравнивание текста в блоках - выравнивание заголовка и текста в блоках;
  • Изменить размер большого изображения - включение изменения размера под собственные / выключение для отображения исходного размера;
  • Размер большого изображения (Ш x В) - для больших экранов;
  • Изменить размер среднего изображения - включение изменения размера под собственные + если изображение для средних экранов не добавлено, то будет создано из большого. Выключение для отображения исходного размера;
  • Размер среднего изображения (Ш x В) - для средних экранов;
  • Изменить размер маленького изображения - включение изменения размера под собственные + если изображение для маленьких экранов не добавлено, то будет создано из большого. Выключение для отображения исходного размера;
  • Размер маленького изображения (Ш x В) - для маленьких экранов.

Доступ:

  • Доступ - кому будет виден этот блок: для всех пользователей / только для зарегистрированных / только для гостей;
  • Группы покупателей - управление доступом к группам клиентов.

Дизайн:

  • Шаблон - после создания шаблона для модуля, вы можете выбрать его здесь.

Стилизация

Модуль имеет четыре вида отображения блоков:

  • cs_html_content_pro - блоки отображаются в виде колонок / каждый блок может быть ссылкой (Шаблон - по умолчанию);
  • cs_html_content_pro_item_with_button - блоки отображаются в виде колонок / ссылки в виде кнопки;
  • cs_html_content_pro_tabs - блоки отображаются в виде вкладок;
  • cs_html_content_pro_accordion - блоки отображаются в виде аккордеона.

Размещение нового шаблона модуля

Создайте новый шаблон модуля и разместите его в папке:

Для успешного создания и использования нового шаблона модуля, следует придерживаться следующих шагов:

  • Создайте новый файл шаблона и сохраните его с расширением, указывающим на его назначение (например, ".twig" или ".tpl").
  • Переместите этот файл в соответствующую папку для модулей:
catalog/view/theme/default/template/custom_templates/module/html_content_pro/имя файла на латинице.(twig.tpl)

После выполнения этих шагов, ваш новый шаблон модуля будет доступен в списке.

Внесение изменений в CSS стили

Если стандартный дизайн не соответствует вашему дизайну сайта, вы можете внести изменения в CSS стилях в файле «cs_html_content_pro.min.css», который находится по следующему пути:

catalog/view/javascript/cs_libs/css/cs_html_content_pro.css
