Документация Дополнительное описание категорий + настройки

Модификация позволяет добавлять текст, баннеры или HTML-код в нижней части страниц категорий, что отличается от стандартного описания. Также она предоставляет управление верхним и нижним описанием, изображениями и скриптами для категорий, предлагая больше настроек для отображения информации.

Мы старались обеспечить максимальную совместимость и удобство использования, но некоторые аспекты могут требовать вашего внимания и настройки. Не стесняйтесь обращаться к нам с любыми вопросами или потребностью в помощи.

Важно!

Модификатор прошел тестирование на стандартном шаблоне Opencart, но может потребовать адаптации для других шаблонов.

При установке создаются дополнительные поля в базе данных. Рекомендуем сделать резервную копию данных перед установкой для возможности восстановления данных в случае необходимости.

Модификатор не устанавливается?

Попробуйте сначала установить «Localcopy OCMOD», а затем попробуйте установить наш модификатор снова. Эта последовательность может разрешить возможные конфликты и поможет вам успешно внедрить наши изменения на вашем сайте.

Установка расширения

Процесс установки модификатора состоит из двух шагов:

  • «AD_Category_x_OpenCart_x.ocmod.zip» - это архив, содержащий основной функционал модуля и модификатора;
  • «AD_Category_FrontEnd_Default_OpenCart_x.ocmod.zip» - это архив, содержащий модификатор для подключения шаблона.

Чтобы установить модификатор, выполните следующие шаги:

  • Войдите в Админ-панель и перейдите в раздел «Меню -> Расширения -> Установка расширений»;
  • Загрузите и установите оба архива;
  • После успешной установки вы получите уведомление об этом;
  • Перейдите в раздел «Меню -> Расширения -> Модификаторы» и в правом верхнем углу нажмите кнопку «Обновить»;

После этого перейдите в раздел «Меню -> Система -> Группы пользователей» и установите права доступа для администратора (в разделах «Разрешить просмотр» и «Разрешить редактирование» выберите «extension/module/cs_ad_category»).

Завершив эти шаги, перейдите в раздел «Меню -> Расширения -> Модули» и установите модуль «Дополнительное Описание Категориям + Настройки».

Настройка расширения

Для настройки расширения перейдите в «Меню -> Расширения -> Модули -> Дополнительное Описание Категориям + Настройки»

Основные параметры настройки:

  • Позиции описаний - позволяет менять местами верхний и нижний опис;
  • Статус - отвечает за активацию этого расширения.

Дополнительные параметры настройки верхнего и нижнего описания:

  • Статус описания - отображение блока с описанием на сайте;
  • Статус изображения - отображает изображение категории рядом с текстом;
  • Позиция изображения - определяет, с какой стороны текста разместить изображение;
  • Описание на страницах пагинации - настраивает отображение текста на первой странице или на всех;
  • Статус кнопки «Подробнее» - позволяет включить или выключить скрипт для всех описаний категорий;
  • Высота видимого текста - регулирует количество видимого текста, остальная часть будет скрыта;
  • Разрешение экрана - настраивает разрешения экрана, на которых этот скрипт будет работать.

Статус БД:

  • Отображение статуса полей в таблице БД;
  • Управление установкой или удалением полей из БД.

Настройка в Категориях

Чтобы настроить расширение для конкретного товара, перейдите в раздел «Меню -> Каталог -> Категории» и выберите нужную категорию. Дополнительные параметры находятся во вкладке «Дизайн»

Здесь вы можете настроить следующие параметры:

  • Статус кнопки «Подробнее» - включите или выключите скрипт для конкретной категории или оставьте по умолчанию, тогда будут применяться настройки из модуля;
  • Высота видимого текста - определите количество текста, которое будет видимым, остальная часть текста будет скрыта.

Эти настройки позволят вам настроить расширение для конкретных категорий товаров в вашем каталоге.

Стилизация

Вы можете изменять верстку блоков с описанием, редактируя файлы, которые находятся по следующим путям:

Верхний блок описания:

/catalog/view/theme/default/template/custom_templates/addons/ad_category_top.(twig/tpl)

Нижний блок описания:

/catalog/view/theme/default/template/custom_templates/addons/ad_category_bottom.(twig/tpl)

Если стандартный дизайн кнопки «Развернуть / Свернуть» не соответствует вашему сайту, вы можете изменить CSS-стили в файле «cs_ad_category.min.css», который находится по следующему пути:

/catalog/view/javascript/cs_libs/css/cs_readmore.min.css
