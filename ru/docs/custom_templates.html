Документация Пользовательские шаблоны

Позволяет установить определенный шаблон для товара, категории, производителя (бренда) или информационных страниц. Вы также можете назначить один шаблон для всех товаров определенной категории и, при необходимости, переопределить этот шаблон для конкретного товара.

Информация пользователям

Пожалуйста, внимательно ознакомьтесь с этой информацией перед началом использования наших продуктов. Мы стремились обеспечить максимальную совместимость и удобство использования, но некоторые аспекты могут требовать вашего внимания и настройки. Не стесняйтесь обращаться к нам с любыми вопросами или запросом помощи.

  • Модификатор прошёл тестирование на стандартном шаблоне Opencart, однако может потребовать адаптации для других шаблонов.
  • При установке создаются дополнительные поля в базе данных. Рекомендуем создать резервную копию данных перед установкой для возможности восстановления данных при необходимости.
  • Обратите внимание, что когда упоминается ТЕМА, речь идет о конкретных файлах шаблона в вашей теме, а не о всей теме. С использованием данного расширения нельзя устанавливать разные темы для каждого элемента.
  • Если вы установили стороннюю тему на Opencart 3.x с расширением файлов «.tpl»,
    модификатор не будет работать.
  • Созданные файлы должны иметь расширение файлов «.twig».

Попробуйте сначала установить «Localcopy OCMOD», а затем попробуйте установить наш модификатор снова. Эта последовательность может разрешить возможные конфликты и поможет вам успешно внедрить наши изменения на вашем сайте.

Загрузить для OpenCart 3x

Установка модификатора

Чтобы установить модификатор, выполните следующие шаги:

  • Войдите в Админ-панель и перейдите в раздел Меню -> Расширения -> Установка расширений;
  • Загрузите и установите архивный файл Custom_Templates_x_Opencart_x.ocmod.zip;
  • После успешной установки перейдите в раздел Меню -> Расширения -> Модификаторы и в правом верхнем углу нажмите кнопку «Обновить».
  • Завершив эти шаги, перейдите в раздел Меню -> Расширения -> Модули и установите модуль Пользовательские шаблоны.

Настройка модификатора

Для настройки модификатора перейдите в раздел Меню -> Расширения -> Модули -> Пользовательские шаблоны и измените статус модификатора на «Включено».

Как установить собственный файл:

  • После установки на сервере автоматически появится папка custom_templates по пути:

    catalog/view/theme/default/template/custom_templates/

    Если вы используете другую тему, обязательно создайте эту папку и внутренние папки category, information, manufacturer, product.

  • Добавьте файл с внесенными изменениями в соответствующую папку (имя файла латиницей, расширение .twig).

Настройки в разделе:

Для настройки расширения для конкретного раздела перейдите в Меню -> Каталог -> (Товары, Категории, Информационные статьи, Производители). Параметры находятся в разделе Дизайн.

  • Выберите нужный шаблон из списка;
  • В категориях назначьте шаблон для всех товаров в этой категории;
  • В товарах выберите или измените шаблон;
  • В производителях и информационных статьях укажите новый шаблон.
