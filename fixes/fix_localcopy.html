Фікс Localcopy - виправлення помилки встановлення ocmod

Localcopy - це виправлення, яке забезпечує правильне завантаження ocmod-модулів через адміністративний розділ сайту на Opencart. Це вирішення проблеми під час завантаження модуля, коли виникає помилка «Папка не дозволена для запису! / Директорія не доступна для запису / The directory is not allowed to be written to!»
