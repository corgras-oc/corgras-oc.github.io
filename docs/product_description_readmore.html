Документація JS Readmore для опису товару

Дозволяє приховувати частину довгого тексту на сторінці товару та надає користувачам можливість розгортати та ховати опис товару за потребою. Цей модифікатор забезпечує більш гнучкий та зручний спосіб подання інформації на сторінках товарів вашого інтернет-магазину. Вам більше не потрібно завантажувати сторінки додатковим вмістом, оскільки користувачі можуть вирішувати, коли їм потрібно переглянути повний опис товару.

Інформація користувачам

Будь ласка, уважно ознайомтеся з цією інформацією перед тим, як розпочати використання наших продуктів. Ми намагалися забезпечити максимальну сумісність та зручність використання, але деякі аспекти можуть вимагати вашої уваги та налаштувань. Не соромтеся звертатися до нас з будь-якими питаннями чи потребою в допомозі.

  • Модифікатор пройшов тестування на стандартному шаблоні Opencart, але може потребувати адаптацій для інших шаблонів.
  • При встановленні створюються додаткові поля в базі даних. Рекомендуємо зробити резервну копію даних перед встановленням для можливості відновлення даних в разі потреби.

Спробуйте спочатку встановити «Localcopy OCMOD», а потім спробуйте встановити наш модифікатор знову. Ця послідовність може вирішити можливі конфлікти та допоможе вам успішно впровадити наші зміни на вашому сайті.

Встановлення розширення

Щоб встановити модифікатор, виконайте наступні кроки:

  • Увійдіть в Адмін-панель та перейдіть до розділу Меню -> Розширення -> Встановлення розширень;
  • Завантажте та встановіть архівний файл PD_Readmore_x_OpenCart_x.ocmod.zip;
  • Після успішного встановлення перейдіть до розділу Меню -> Розширення -> Модифікатори і у правому верхньому куті натисніть кнопку «Оновити»;
  • Після перейдіть до розділу Меню -> Розширення -> Модулі і встановіть модуль JS Readmore Для Опису Товару.

Налаштування розширення

Для налаштування розширення перейдіть Меню -> Розширення -> Модулі -> JS Readmore Для Опису Товару

Основні параметри налаштування:

  • Статус - відповідає за активацію цього розширення;
  • Статус JavaScript для всіх товарів - дозволяє увімкнути або вимкнути скрипт для всіх описів товарів;
  • Висота при згорнутому стані - регулює кількість видимого тексту, приховуючи решту;
  • Роздільна здатність екрану - налаштовує роздільні здатності екрану, на яких цей скрипт буде працювати.

Статус БД:

  • Відображення статусу полів у таблиці БД;
  • Керування установкою або видаленням полів з БД.

Налаштування у товарах

Щоб налаштувати розширення для конкретного товару, перейдіть в розділ Меню -> Каталог -> Товари і оберіть потрібний товар. Додаткові параметри знаходяться у вкладці Дизайн

Тут ви можете налаштувати наступні параметри:

  • Статус JavaScript для товару - увімкніть або вимкніть скрипт для конкретного товару, або залиште за замовчуванням, тоді будуть застосовуватися налаштування з модуля;
  • Висота при згорнутому стані - визначте кількість тексту, який буде видимим, інша частина тексту буде прихована.

Ці налаштування допоможуть вам налаштувати розширення для конкретних товарів.

Стилізація

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не відповідає вашому сайту, ви можете змінити CSS стилі в файлі cs_readmore.min.css, який розташований за таким шляхом:

/catalog/view/javascript/cs_libs/css/cs_readmore.min.css
