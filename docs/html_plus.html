Документація Текстовий блок HTML PLUS

Дозволяє ефективно керувати скриптом «Детальніше», вирішувати, на яких екранах він має функціонувати, налаштовувати рівні заголовків, створювати власні шаблони, керувати доступом до контенту, легко вставляти зображення та відео, а також регулювати розташування контенту відносно зображень чи відео. Це робить модуль потужним інструментом для створення гнучкого та інтуїтивно зрозумілого контенту на вашому веб-сайті.

Встановлення модуля

Щоб встановити модуль, виконайте наступні кроки:

  • Увійдіть в Адмін-панель та перейдіть до розділу Меню -> Розширення -> Встановлення розширень;
  • Завантажте та встановіть архівний файл HTML_Content_Plus_x_OpenCart_x.ocmod.zip;
  • Після успішного встановлення перейдіть до розділу Меню -> Система -> Групи користувачів і встановіть права доступу для адміністратора (в розділах «Дозволити перегляд» та «Дозволити редагувати» оберіть extension/module/cs_html_content_plus);
  • Завершивши ці кроки, перейдіть до розділу Меню -> Розширення -> Модулі і встановіть модуль Текстовий блок HTML PLUS.

Налаштування модуля

Для налаштування модуля перейдіть Меню -> Розширення -> Модулі -> Текстовий блок HTML PLUS

Основні параметри налаштування:

  • Назва модуля - назва, яка відображатиметься в адмін-панелі;
  • Статус - відповідає за увімкнення модуля;
  • Зображення - вставка зображень для різних розмірів екранів або одного для всіх;
  • Відео список - вибір, звідки буде вставлено відео;
  • Посилання на відео - для сервісів, що підтримуються, достатньо вставити будь-яке посилання на відео, для інших - скопіювати посилання з iframe src="".

Додаткові параметри налаштування:

  • Позиція контенту - розташування тексту щодо зображення чи відео;
  • Рівень заголовка - рівень заголовка для відображення на сайті;
  • Статус скрипта «Детальніше» - увімкнення / вимкнення скрипту;
  • Висота видимого тексту - яка буде читана, а інша частина тексту буде прихована;
  • Розширення екрану - на яких дозволах екрану працюватиме скрипт;
  • Змінити розмір великого зображення - увімкнення зміни розміру під власні / вимкнення для відображення вихідного розміру;
  • Розмір великого зображення (Ш x В) - для великих екранів;
  • Змінити розмір середнього зображення - увімкнення зміни розміру під власні + якщо зображення для середніх екранів не додано, то буде створено з великого. Вимкнення для відображення вихідного розміру;
  • Розмір середнього зображення (Ш x В) - для середніх екранів;
  • Змінити розмір маленького зображення - увімкнення зміни розміру під власні + якщо зображення для маленьких екранів не додано, то буде створено з великого. Вимкнення для відображення вихідного розміру;
  • Розмір маленького зображення (Ш x В) - для маленьких екранів.

Доступ:

  • Доступ - кому буде видно цей блок: для всіх користувачів / тільки для зареєстрованих / тільки для гостей;
  • Групи покупців - управління доступом до груп клієнтів.

Дизайн:

  • Шаблон - після створення шаблону для модуля, можете вибрати його тут.

Стилізація

Шаблони:

Для налаштування вигляду блоків з описом, cтворіть власний шаблон для вашого модуля та розмістіть його в папці:

catalog/view/theme/default/template/custom_templates/module/html_content_plus/ім'я_файлу_латинськими_літерами.twig.tpl

Як альтернативу, ви можете використовувати стандартний шаблон за таким шляхом:

catalog/view/theme/default/template/extension/module/cs_html_content_plus.twig.tpl

CSS-стилізація:

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не відповідає вашому дизайну сайту, ви можете внести зміни у CSS стилях в файлі cs_html_content_plus.min.css, який розташований за таким шляхом:

catalog/view/javascript/cs_libs/css/cs_html_content_plus.min.css
