Документація Текстовий блок HTML PRO

Дозволяє створювати та налаштовувати сітки текстових блоків на вашому веб-сайті. За допомогою цього інструменту ви можете легко створювати необмежену кількість текстових блоків та налаштовувати їх відповідно до ваших потреб і дизайну. Повна гнучкість та контроль над розташуванням і вмістом тексту на вашому веб-сайті.

Встановлення модуля

Щоб встановити модуль, виконайте наступні кроки:

  • Увійдіть в Адмін-панель та перейдіть до розділу Меню -> Розширення -> Встановлення розширень;
  • Завантажте та встановіть архівний файл HTML_Content_Pro_x_OpenCart_x.ocmod.zip;
  • Після успішного встановлення перейдіть до розділу Меню -> Система -> Групи користувачів і встановіть права доступу для адміністратора (в розділах «Дозволити перегляд» та «Дозволити редагувати» оберіть extension/module/cs_html_content_pro);
  • Завершивши ці кроки, перейдіть до розділу Меню -> Розширення -> Модулі і встановіть модуль Текстовий блок HTML PRO.

Налаштування модуля

Для налаштування модуля перейдіть Меню -> Розширення -> Модулі -> Текстовий блок HTML PRO

Основні параметри налаштування:

  • Назва модуля - назва, яка відображатиметься в адмін-панелі;
  • Статус - статус модуля відповідає за відображення модуля на сайті.

Розділ:

  • Зображення - вставляння зображень для різних розмірів екранів або одне для всіх;
  • Відео список - вибір звідки буде вставлено відео;
  • Посилання на відео - для сервісів, що підтримуються, достатньо вставити будь-яке посилання на відео для інших скопіювати посилання з iframe src="".

Блоки:

  • Зображення - вставляння зображень для різних розмірів екранів або одне для всіх;
  • Class іконки - щоб відобразити іконку, додайте її CLASS. Дивитись документацію до бібліотеки, яка підключена.;
  • Відео список - вибір звідки буде вставлено відео;
  • Посилання на відео - для сервісів, що підтримуються, достатньо вставити будь-яке посилання на відео для інших скопіювати посилання з iframe src="";
  • Позиціювання медіа-елементів - розташування медіа-елементів щодо тексту;
  • Категорія - посилання на категорію;
  • Стаття - посилання на інформаційну статтю;
  • Виробник - посилання на виробника;
  • Інше посилання - будь-яке інше посилання;
  • Назва кнопки - текст, який відображатиметься на кнопці;
  • Class - індивідуальний CSS клас для блоку;
  • Id - індивідуальний CSS ідентифікатор для блоку;
  • Атрибути - індивідуальний HTML атрибут для блоку.

Налаштування:

  • Сітка контенту - в якому порядку відображатимуться блоки;
  • Class - індивідуальний CSS клас для розділу;
  • Id - індивідуальний CSS ідентифікатор для розділу;
  • Атрибути - індивідуальний HTML атрибут для розділу;
  • Колонки - налаштування кількості колонок для різних екранів;
  • Вирівнювання тексту в блоках - вирівнювання заголовка та тексту в блоках;
  • Змінити розмір великого зображення - увімкнення зміни розміру під власні / вимкнення для відображення вихідного розміру;
  • Розмір великого зображення (Ш x В) - для великих екранів;
  • Змінити розмір середнього зображення - увімкнення зміни розміру під власні + якщо зображення для середніх екранів не додано, то буде створено з великого. Вимкнення для відображення вихідного розміру;
  • Розмір середнього зображення (Ш x В) - для середніх екранів;
  • Змінити розмір маленького зображення - увімкнення зміни розміру під власні + якщо зображення для маленьких екранів не додано, то буде створено з великого. Вимкнення для відображення вихідного розміру;
  • Розмір маленького зображення (Ш x В) - для маленьких екранів.

Доступ:

  • Доступ - кому буде видно цей блок: для всіх користувачів / тільки для зареєстрованих / тільки для гостей;
  • Групи покупців - управління доступом до груп клієнтів.

Дизайн:

  • Шаблон - після створення шаблону для модуля, можете вибрати його тут.

Стилізація

Модуль має чотири види відображення блоків:

  • cs_html_content_pro - блоки відображаються у вигляді колонок / кожен блок може бути посиланням (Шаблон - за замовчуванням);
  • cs_html_content_pro_item_with_button - блоки відображаються у вигляді колонок / посилання у вигляді кнопки;
  • cs_html_content_pro_tabs - блоки відображаються у вигляді вкладок;
  • cs_html_content_pro_accordion - блоки відображаються у вигляді акордеону.

Розміщення нового шаблону модуля

Створіть новий шаблон модуля та розмістіть його у папці:

Для успішного створення та використання нового шаблону модуля, слід дотримуватися наступних кроків:

  • Створіть новий файл шаблону та збережіть його з розширенням, що вказує на його призначення (наприклад, ".twig" або ".tpl").
  • Перемістіть цей файл у відповідну папку для модулів:
catalog/view/theme/default/template/custom_templates/module/html_content_pro/ім'я файлу на латиниці.(twig.tpl)

Після виконання цих кроків, ваш новий шаблон модуля буде доступний в списку.

Внесення змін у CSS стили

Якщо стандартний дизайн не відповідає вашому дизайну сайту, ви можете внести зміни у CSS стилях в файлі cs_html_content_pro.min.css, який розташований за таким шляхом:

catalog/view/javascript/cs_libs/css/cs_html_content_pro.css
