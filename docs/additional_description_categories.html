Ми намагалися забезпечити максимальну сумісність та зручність використання, але деякі аспекти можуть вимагати вашої уваги та налаштувань. Важливо!

Модифікатор пройшов тестування на стандартному шаблоні Opencart, але може потребувати адаптацій для інших шаблонів.

При встановленні створюються додаткові поля в базі даних. Рекомендуємо зробити резервну копію даних перед встановленням для можливості відновлення даних в разі потреби.

Модифікатор не встановлюється?

Спробуйте спочатку встановити «Localcopy OCMOD», а потім спробуйте встановити наш модифікатор знову. Ця послідовність може вирішити можливі конфлікти та допоможе вам успішно впровадити наші зміни на вашому сайті.

Встановлення розширення

Процес встановлення модифікатора складається з двох кроків:

  • «AD_Category_x_OpenCart_x.ocmod.zip» - цей архів містить основний функціонал модуля та модифікатора;
  • «AD_Category_FrontEnd_Default_OpenCart_x.ocmod.zip» - цей архів містить модифікатор для підключення шаблону.

Щоб встановити модифікатор, виконайте наступні кроки:

  • Увійдіть в Адмін-панель та перейдіть до розділу «Меню -> Розширення -> Встановлення розширень»;
  • Завантажте та встановіть обидва архіви;
  • Після успішного завершення встановлення отримаєте повідомлення про це;
  • Перейдіть до розділу «Меню -> Розширення -> Модифікатори» і у правому верхньому куті натисніть кнопку «Оновити»;

Після цього перейдіть до «Меню -> Система -> Групи користувачів» і встановіть права доступу для адміністратора (в розділах «Дозволити перегляд» та «Дозволити редагувати» оберіть «extension/module/cs_ad_category»).

Завершивши ці кроки, перейдіть до розділу «Меню -> Розширення -> Модулі» і встановіть модуль «Додатковий Опис Категоріям + Налаштування».

Налаштування розширення

Для налаштування розширення перейдіть «Меню -> Розширення -> Модулі -> Додаткове Опис Категоріям + Налаштування»

Основні параметри налаштування:

  • Позиції описів - дозволяє змінювати місцями верхній та нижній опис;
  • Статус - відповідає за активацію цього розширення.

Додаткові параметри налаштування верхнього та нижнього опису:

  • Статус опису - відображення блоку з описом на сайті;
  • Статус зображення - відображає зображення категорії поруч із текстом;
  • Позиція зображення - визначає, з якого боку тексту розташовувати зображення;
  • Опис на сторінках пагінації - налаштовує відображення тексту на першій сторінці чи на всіх;
  • Статус кнопки «Детальніше» - дозволяє включити або вимкнути скрипт для всіх описів категорій;
  • Висота видимого тексту - регулює кількість видимого тексту, а іншу частину приховує;
  • Розширення екрану - налаштовує роздільні здатності екрану, на яких цей скрипт працюватиме.

Статус БД:

  • Відображення статусу полів у таблиці БД;
  • Керування установкою або видаленням полів з БД.

Налаштування у категоріях

Щоб налаштувати розширення для конкретного товару, перейдіть в розділ «Меню -> Каталог -> Категорії» і оберіть потрібну категорію. Додаткові параметри знаходяться у вкладці «Дизайн»

Тут ви можете налаштувати наступні параметри:

  • Статус кнопки «Детальніше» - увімкніть або вимкніть скрипт для конкретної категорії, або залиште за замовчуванням, тоді будуть застосовуватися налаштування з модуля;
  • Висота видимого тексту - визначте кількість тексту, який буде видимим, інша частина тексту буде прихована.

Ці налаштування допоможуть вам налаштувати розширення для конкретних категорій товарів у вашому каталозі.

Стилізація

Ви можете змінювати верстку блоків з описом, редагуючи файли, які розташовані за такими шляхами:

Верхній блок опису:

/catalog/view/theme/default/template/custom_templates/addons/ad_category_top.(twig/tpl)

Нижній блок опису:

/catalog/view/theme/default/template/custom_templates/addons/ad_category_bottom.(twig/tpl)

Якщо стандартний дизайн кнопки «Розгорнути / Згорнути» не відповідає вашому сайту, ви можете змінити CSS стилі в файлі «cs_ad_category.min.css», який розташований за таким шляхом:

/catalog/view/javascript/cs_libs/css/cs_ad_category.min.css
