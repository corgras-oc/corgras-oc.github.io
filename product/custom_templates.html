Про модифікатор Користувальницькі шаблони

Дозволяє встановити певний шаблон для товару, категорії, виробника (бренду) або інформаційних сторінок. Ви також можете призначити один шаблон для всіх товарів певної категорії і, за потреби, перевизначити цей шаблон для конкретного товару.

Можливості модифікатора

  • Призначити певний шаблон для товару, категорії, виробника (бренду) чи інформаційних сторінок;
  • Призначити один шаблон для всіх товарів під певною категорією;
  • Перевизначити шаблон для конкретного товару.
