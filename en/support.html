Technical support

  • Technical support is provided only for official module purchases;
  • In some cases, technical support may not be possible without access to the admin panel and FTP. Access details should include the host, login, and password. If your port is different from the standard 21, please specify it. If there is IP protection on your hosting, disable it during the support session to allow connection;
  • If you have any questions or suggestions;
  • If the module lacks functionality that you need, the author may decline customization for you. This would involve additional paid work. In such a case, you should create a precise technical task and submit it to support for review;
  • Do not duplicate messages across different communication channels; it does not expedite the support process;
  • If you have already contacted support, please await a response. Response times can vary from minutes to a couple of days.
