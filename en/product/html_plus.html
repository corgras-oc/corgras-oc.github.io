About the module HTML-Content PLUS

Allows for efficient management of the «Read More» script, determining on which screens it should operate, adjusting header levels, creating custom templates, controlling access to content, easily embedding images and videos, and adjusting the placement of content relative to images or videos. This makes the module a powerful tool for creating flexible and user-friendly content on your website.

Module documentation

Module features

  • Script control for «Details»;
  • Control over which screen permissions the script will work on;
  • Heading level (H1-H6) management;
  • Utilizing a custom template for the module;
  • Access control to content;
  • Inserting images for different screen sizes;
  • Image size control;
  • Deferred image loading using browser capabilities;
  • Inserting videos;
  • Positioning content relative to images or videos;
  • Multilingual support.
